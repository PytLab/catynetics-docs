plot_all(**kwargs)[source]¶

Plot energy profile for all elementary reactions.

Parameters:
  • energies (tuple or list) -- energies for states of a elementary reaction
  • n (int) -- the point number in each state, default is 100.
  • hline_leng -- the length of the horizontal line for the IS and FS
  • peak_width (float) -- the width of the peak in energy profile, default is 1.0.
  • interp_method (str) -- the type of interpolation algorithm, possible value: "spline", "quadratic", default is "spline".
  • rxn_equation (str) -- elementary reaction equation, default is None.
  • line_width (float) -- line width, default is 3.
  • color (str) -- color code of the line, default is #000000 (black).
  • shadow_color (str) -- color code of the shadow lines, default is #595959.
  • shadow_depth (int) -- shadow depth of the line, default is 0, no shadow.
