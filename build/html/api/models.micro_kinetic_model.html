models.MicroKineticModel¶

class scaks.models.micro_kinetic_model.MicroKineticModel(**kwargs)[source]¶

Bases: scaks.models.kinetic_model.KineticModel

Class for micro-kinetic model.

Parameters:
  • setup_file (int) -- kinetic model set up file
  • setup_dict (dict) -- A dictionary contains essential setup parameters for kinetic model.
  • logger_level (int) -- logging level
  • file_handler_level (int) -- logging level for file handler
  • console_handler_level (int) -- logging level for console handler
Returns:

None

Example:

>>> from scaks.models.kinetic_model import MicroKineticModel
>>> model = MicroKineticModel(setup_file="setup.mkm", logger_level=30)
Attributes:

decimal_precision (int): Data precision, default value is 100

perturbation_size (float): Perturbation size for numerical jacobian matrix, default value is 0.01

perturbation_direction (str): Direction of perturbation, left or right, default is 'right'

archived_variables (str): Archived variables. Possible values: 'initial_guess', 'steady_state_coverages', 'steady_state_error', 'rates', 'net_rates', 'reversibilities', 'tofs'

numerical_representation (str): Numerical representation method, value could be 'mpmath' or 'sympy'.

rootfinding(str): Rootfinding iterator type, default value is 'MDNewton', possible value can be 'MDNewton' or 'ConstrainedNewton'

tolerance (float): Iteration tolerance, default is 1e-8

max_rootfinding_iterations (int): Max iteraction steps, default is 100

ode_buffer_size (int): Ode integration buffer size, default is 500

ode_output_interval (int): Ode ouptut interval, default is 200

data_file (str): Filename for data store, default is 'data.pkl'

log_allowed (bool): If log output is allowed, default is True

TOFs (list of float): Turnover frequencies calculated.

reversibilities (list of float): Reversibilities calculated.

error (list of float): Residual errors after iterations.

analysis (list of scaks.plugins.analysis.OnTheFlyAnalysis: On-the-fly analysis plugins

run(**kwargs)[source]¶

Run function to solve Micro-kinetic model using Steady State Approxmiation to get steady state coverages and turnover frequencies (TOF).

Parameters:
  • init_cvgs (list of float) -- Initial guess for coverages
  • relative_energies (dict) -- Relative energies for all elementary reactions.

Note

keys Gaf and Gar must be in relative energies dict. e.g. {"Gaf": [...], "Gar": [...], "dG": [...]}

Parameters:
  • fsolve (bool) -- use scipy.optimize.fsolve to get low-precision root or not
  • coarse_guess (bool) -- use fsolve to do initial coverages preprocessing or not
  • XRC (bool) -- calculate degree of rate control or nor
  • epsilon (float) -- the change of energy for XRC calculation, default is 10-5.
  • product_name (str) -- Production name of the model. e.g. 'CH3OH_g'
Returns:

None
