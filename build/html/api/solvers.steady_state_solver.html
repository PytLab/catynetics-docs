static get_kCT(Ea, Auc, act_ratio, m, T, p=101325, f=1.0)¶

Static function to get rate constant/collision rate according to Collision Theory.

Parameters:
  • Ea (float) -- energy barrier( NOT free energy barrier )
  • Auc (float) -- area of unitcell (m^-2)
  • act_ratio (float(<=1.0)) -- area of active sites/area of unitcell
  • p (float) -- pressure of gas, default value is 101325 Pa (atm).
  • m (float) -- absolute mass of molecule (kg)
  • f (float(<= 1.0)) -- factor accounts for a further reduction in the sticking probability, if particle with certain initial states are not efficiently steered along the MEP, and reflected by a higher barrier
  • T (float) -- temperature (K)

Solve the steady state equations using ODE integration.

Parameters:algo (str) -- algorithm for ODE solving, optional, possible values: 'vode' | 'zvode' | 'lsoda' | 'dopri5' | 'dop853'

Note

for more details of integration algorithm, see: https://docs.scipy.org/doc/scipy/reference/generated/scipy.integrate.ode.html

Parameters:
  • time_span (float) -- time span for each step, default to be 0.1
  • time_start (float) -- time when begin integration
  • time_end (float) -- time when stop integration
  • initial_cvgs (tuple of float) -- initial coverages at time_start
  • relative_energies (dict) -- Relative energies for calculation, if not provided, use model's relative energies, default is None

Note

keys "Gaf and G:obj:Gar must be in relative energies dict

Parameters:traj_output (bool) -- output ODE integration trajectory or not, default value is False.
Returns:the integrated time
Return type:float
Returns:integrated function values
Return type:list of float
Examples::
>>> m.solver.solve_ode(initial_cvgs=(0.0, 0.0))
