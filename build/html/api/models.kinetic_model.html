models.KineticModel¶

class scaks.models.kinetic_model.KineticModel(**kwargs)[source]¶

The base class for kinetic models.

Parameters:
  • setup_file (str) -- kinetic model set up file
  • setup_dict (dict) -- A dictionary contains essential setup parameters for kinetic model
  • logger_level (int) -- logging level for logger
  • file_handler_level (int) -- logging level for file handler
  • console_handler_level (int) -- logging level for console handler

Example:

>>> from scaks.models import KineticModel
>>> model = KineticModel(setup_file="setup.mkm", logger_level=30)
Attributes:

setup_file(str): kinetic model setup file

setup_dict(dict): A dictionary contains essnetial setup parameters for kinetic model

logger_level(int): The logging level for built-in logger

file_handler_level(int): The logging level for file handler

console_handler_level(int): The logging level for console handler

parser(object): Kinetic Model parser, possible type could be scaks.parsers.RelativeEnergyParser, scaks.parsers.AbsoluteEnergyParser, scaks.parsers.KMCParser

solver(object): Kinetic Model solver, possible type could be scaks.solvers.SteadyStateSolver, scaks.solvers.KMCSolver

corrector(object): Kinetic Model data corrector, possible type could be ThermodynamicCorrector

temperature(float): Environment temperature in K, default is 298.0K

rxn_expressions(list of str): Reaction expressions for elementary reactions, default is []

species_definitions(dict): All species definition in kinetic model after parsing

rate_algo(str): Algorithm for rate calculation, could be 'TST' for transition state theory or 'CT' for collision theory, default value is 'TST'
