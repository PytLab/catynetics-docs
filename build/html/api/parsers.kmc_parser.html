static get_molecular_mass(species_name, absolute=False)¶

Static function to get relative/absolute molecular mass.

Parameters:
  • species_name -- name of the molecule species
  • absolute (bool) -- return absolute mass or not(default)
Example::
>>> m.parser.get_molecular_mass('CH4')
>>> 16.04246
>>> m.parser.get_molecular_mass('CH4', absolute=True)
>>> 2.6639131127638393e-26
